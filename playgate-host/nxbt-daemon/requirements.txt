# PlayGate NXBT daemon dependencies
#
# The daemon uses nuxbt — an actively maintained fork of NXBT. Pinned to our
# own fork's deploy/playgate branch, which combines two changes the daemon
# relies on (each also opened as a standalone PR against upstream):
#   - feat/event-driven-send-loop: select()-driven send loop (lower input latency)
#   - fix/direct-input-plus-minus-swap: correct Plus/Minus bit packing
# The Plus/Minus fix is why nxbtd.py no longer compensates for the swap; if you
# repoint this at a nuxbt without that fix, restore the compensation first.
# Switch 2 controller emulation is on the roadmap but not yet implemented.
#
# NOTE: nuxbt only runs on Linux with BlueZ ≥ 5.50 and Python 3.11+.
# Do NOT install on macOS or Windows — it requires DBus and BlueZ.
#
# System prerequisites (Debian/Ubuntu/Raspberry Pi OS):
#   sudo apt install git python3-dbus python3-gi
# (python3-dbus provides `dbus`, python3-gi provides `gi` — the only
# third-party modules nuxbt's headless control path imports. Installing
# them via apt avoids compiling dbus-python/PyGObject from source.)
#
# Install on Raspberry Pi / Debian / Ubuntu:
#   sudo pip3 install --break-system-packages -r requirements.txt
# or in a virtualenv (use --system-site-packages so dbus/gi are visible):
#   python3 -m venv --system-site-packages venv && source venv/bin/activate
#   pip install -r requirements.txt
