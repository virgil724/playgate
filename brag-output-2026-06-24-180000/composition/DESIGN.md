# DESIGN: PlayGate Brag Video

## Concept
A polished 20-second launch video for PlayGate — the platform that lets Twitch viewers play a streamer's real Nintendo Switch remotely over WebRTC.

## Visual Identity
- **Background:** Deep blue-black (#0d0f14)
- **Accent:** Bright blue (#4f8cff)
- **Secondary accent:** Teal (#2dd4bf)
- **Text:** Near-white (#e6e9ef)
- **Panels:** Dark gray panels with subtle borders
- **Twitch purple:** #9147ff for Twitch-related elements
- **Status colors:** Green (#22c55e) for connected, amber (#f59e0b) for queue

## Typography
- **Display:** Inter, 800 weight, tight letter-spacing
- **Mono:** JetBrains Mono for codes, status, and technical labels
- **Body:** Inter, clean and readable

## Scene Flow
1. **Hook** (0-3s) — Typing animation: "What if your Twitch chat could play your game?"
2. **Twitch Chat** (3-7s) — Chat overlay with !play command and whispered token code
3. **Redeem** (7-11s) — Token code entry, queue position, connection established
4. **Gamepad** (11-16s) — Live game stream + virtual gamepad with button presses and countdown
5. **Logo** (16-20s) — "PlayGate" + tagline "Your chat doesn't just watch anymore."

## Motion
- Polished, confident reveals with `power2.out` easing
- Sequential typing animations for hook and code entry
- Card-slide reveals for whisper notification
- Gamepad buttons light up sequentially with glow effects
- Subtle audio-reactive glow on accent elements

## Audio
- Music: vol-12 (steady, clean) at 0.35 volume, fading out in final 2s
- SFX: keyboard typing, card slides, interface clicks, impact bell for "Connected" moment
- Beat-locked moments at 9.29s (connected bell) and 17.47s (logo slam)
